Produzieren Sie so viele Teile wie Sie können.



Für ein fertig produziertes Teil müssen folgende Schritte ausgeführt werden.

1. Schritt: Nehmen Sie eine Box mit Polgehäuse (c) aus dem Regal (a).

2. Schritt: Platzieren Sie das Polgehäuse in der Presse (d).

3. Schritt: Platzieren Sie eine Druckscheibe (b) auf dem Polgehäuse.

4. Schritt: Schließen Sie die Presse.

5. Schritt: Entnehmen Sie das fertig produzierte Teil aus der Presse und legen Sie es zurück in die Box.

6. Schritt: Schieben Sie die Box durch die mit "Abgabe" beschriftete Öffnung hinter der Arbeitsfläche.
